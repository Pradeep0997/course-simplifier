# 📚 Course Content Simplification Agent


Course Simplifier is a smart web application that helps students easily understand and organize course content.  
It uses **IBM Natural Language Understanding (NLU)** to analyze course materials and **Cloudant** as a flexible NoSQL database.  
Built with **Node.js** and **EJS**, it offers a simple and clean user experience.

---

## 🚀 Features

✅ Analyze complex course content using IBM NLU  
✅ Store and manage data with IBM Cloudant  
✅ Clean and dynamic web pages rendered with EJS  
✅ Fast and lightweight backend with Node.js

---

## 💻 Tech Stack

- **Backend:** Node.js  
- **Frontend:** EJS Templates  
- **Database:** IBM Cloudant  
- **AI:** IBM Natural Language Understanding (NLU)

---

## ⚙️ Getting Started

1️⃣ **Clone the repository**

```
git clone https://github.com/your-username/course-simplifier.git
2️⃣ Install dependencies


npm install
3️⃣ Set up IBM credentials
Add your IBM NLU and Cloudant credentials to your .env file.

4️⃣ Run the app


node app.js
Visit http://localhost:3000 to view it in your browser!


```
